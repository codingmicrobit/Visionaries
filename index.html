How does it work?

  • GPS - NEO-6M

    The NEO-6GM GPS module is a variant of the NEO-6 series by u-blox. It works by receiving signals from GPS satellites to calculate its location, providing accurate positioning data for various applications.

  • Camara - ESP32-CAM

    The ESP32-CAM module is based on the ESP32 microcontroller and features a camera. It works by running code to capture images or video and can communicate over Wi-Fi for remote monitoring or control.

  • AI and connection with vibration motors - ESP-32

    The ESP32 receives data from the ESP32-CAM, which captures images, identifies motion or objects, and sends this data. The ESP32 controls vibration motors, enabling applications such as security systems or haptic feedback installations.
